Cubikki is a voxel-based rendering project created in PB Engine that integrates advanced graphics techniques using Vulkan. It features physically based rendering (PBR), skeletal animation support, and custom post-processing effects like tonemapping. The project includes a Lua scripting system powered by the Sol library, enabling dynamic gameplay logic and user customization. Designed for performance and flexibility, Cubikki leverages procedural generation and optimized rendering pipelines to deliver high-quality visuals in a voxel-based environment.
